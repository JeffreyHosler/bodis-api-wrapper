Top Level Domains¶

Functions for the top Level Level Domains endpoint. See the Official Documentation for more information.

List Top Level Domains¶

Use this endpoint to obtain a list of top-level domains supported by Bodis.

1Bodis::listTlds();

You can pass any parameters listed in their documentation as an array.

1Bodis::listTlds();
2   'page' => 2,
3   'per_page' => 25
4]);
